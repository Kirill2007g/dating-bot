# AnketaBot

Телеграм-бот для знакомств. Пользователи создают анкеты, листают ленту и получают взаимные матчи.

**Стек:** Python 3.11+, aiogram 3, SQLite

---

## Функционал

### Регистрация

Пошаговый сценарий с валидацией на каждом шаге:

1. Возраст — только число, диапазон 13–99
2. Пол
3. Кого ищет
4. Город — нормализуется к единому написанию (Київ → Киев, Питер → Санкт-Петербург и т.д.)
5. Имя — 2–50 символов
6. Описание — до 500 символов, можно пропустить
7. Медиа — 1–3 файла (фото, видео, кружочек)

После заполнения показывается превью анкеты, можно подтвердить или начать заново.

---

### Лента анкет

Кнопка **Смотреть анкеты** открывает ленту. Алгоритм подбора:

- Фильтрация по городу и предпочтениям по полу
- В 90% случаев показываются анкеты точно такого же возраста, иначе расширяется до ±4 лет
- Уже оцененные анкеты больше не показываются
- Анкеты с активным Premium отображаются выше остальных
- Лимит просмотров — 500 анкет в день (для обычных пользователей)

Действия в ленте:

| Кнопка | Действие |
|--------|----------|
| ❤️ | Лайк |
| 💌 | Лайк с текстовым сообщением |
| 👎 | Дизлайк |
| ↩️ | Отменить последний дизлайк *(только Premium)* |
| 💤 | Выйти в меню |

---

### Лайки и матчи

- При лайке второй пользователь получает уведомление с кнопкой «Посмотреть 👀»
- Если лайк взаимный — оба получают контакт друг друга (матч)
- Перед просмотром обычной ленты бот сначала показывает тех, кто уже лайкнул вас
- Лайк с сообщением (💌): текст добавляется к уведомлению о лайке

---

### Управление профилем

Из главного меню доступно:

- **Моя анкета** — посмотреть свою анкету как её видят другие
- **Заполнить заново** — пересоздать анкету с нуля, сохраняя язык и статус Premium
- **Изменить фото/видео** — загрузить новые медиафайлы
- **Изменить текст** — обновить описание без пересоздания анкеты

---

### Premium

Активируется через Telegram Stars, срок продлевается от текущей даты истечения (не теряется).

| Длительность | Цена |
|-------------|------|
| 2 дня | ⭐ 150 |
| 10 дней | ⭐ 350 |
| 30 дней | ⭐ 500 |
| 90 дней | ⭐ 1000 |

Что даёт Premium:

- Безлимитный просмотр анкет (вместо 500/день)
- Приоритет в ленте у других пользователей
- Кнопка ↩️ — отмена последнего дизлайка

---

### Языки

Поддерживаются русский, украинский и английский. Язык меняется в настройках, интерфейс переключается сразу.

---

### Прочее

- При блокировке бота анкета автоматически скрывается из ленты, при разблокировке — возвращается
- Раз в час всем активным пользователям приходит случайный совет по безопасному общению
- Кнопка «Пожаловаться» доступна после матча

## Структура проекта

```
bot.py                  — запуск, роутеры, обработка платежей, фоновые рассылки
config.py               — BOT_TOKEN из переменной окружения
db.py                   — фасад над репозиториями
states.py               — FSM-состояния диалогов
strings.py              — все тексты и переводы
keyboards.py            — reply и inline клавиатуры
utils.py                — отправка медиа (фото/видео/кружочки, одиночные и группы)

handlers/
  start.py              — регистрация
  profile.py            — управление анкетой, настройки, Premium
  viewing.py            — лента, лайки, матчи

repositories/
  schema.py             — создание таблиц, миграции колонок
  sqlite_client.py      — подключение к БД
  profile_repo.py       — CRUD профилей, алгоритм подбора, лимиты просмотров
  media_repo.py         — медиафайлы профиля
  likes_repo.py         — лайки, дизлайки, проверка матча
  premium_repo.py       — активация и проверка Premium

services/
  media_input.py        — извлечение file_id из входящего сообщения
  media_flow.py         — шаг сбора медиа (регистрация / редактирование)
  profile_format.py     — форматирование подписи анкеты
```

## Запуск

```bash
pip install -r requirements.txt
BOT_TOKEN=your_token python bot.py
```

База данных `dating.db` создаётся автоматически при первом запуске.

## Геокодинг городов

При регистрации город нормализуется через **Nominatim (OpenStreetMap)** — любой вариант написания приводится к каноническому русскому названию:

- `Київ` → `Киев`
- `Питер`, `СПБ` → `Санкт-Петербург`
- `Харків` → `Харьков`
- любой населённый пункт мира — без ручного словаря

Если геокодер недоступен — город сохраняется как ввёл пользователь (`.title()`).

Используется `geopy` + `aiohttp` (асинхронный запрос, не блокирует бота).
